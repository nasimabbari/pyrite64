## 🖥️ What is pyrite64?

pyrite64 is a game engine designed for creating games on the Nintendo 64 console. It also provides an editor so you can build game scenes and levels. The engine uses libdragon, an open-source library that helps developers work on the N64 hardware, and tiny3d, a library for drawing 3D graphics.

This software lets you experiment with game creation in a user-friendly way. You can build, edit, and run simple 3D scenes made for the N64.

## 🛠️ Using pyrite64

Once the program is running, you can start using it to create N64 game scenes. Here are some basic actions:

- **Create a new project**  
  Click `File` > `New` to start a fresh scene.

- **Add objects**  
  Use the editor tools to add 3D models like cubes, spheres, or imported items.

- **Edit properties**  
  Select objects in the scene and adjust their position, rotation, and scale.

- **Apply textures**  
  Add colors or images to objects to make them look like game elements.

- **Test your scene**  
  Use the built-in preview option to render the scene as it would appear on N64 hardware.

- **Save your project**  
  Choose `File` > `Save` to store your work for future editing.
